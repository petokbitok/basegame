# Poker AI Game

Texas Hold'em poker game with AI opponents on Base blockchain.

## Quick Start

```bash
npm install
npm run dev
```

## Setup

1. Deploy contract to Base mainnet via Remix
2. Update `.env`:
```bash
VITE_CDP_PROJECT_ID=your_project_id
VITE_LEADERBOARD_CONTRACT_ADDRESS=0x...
```

## Tech Stack

- React 19 + Vite
- Wagmi v2 + OnchainKit
- Base Account
- Solidity 0.8.20

## Build

```bash
npm run build
```
